# acidentes-rodoviarios-brasil-2017
Projeto de análise de dados sobre acidentes em rodovias federais do Brasil em 2017, desenvolvido com Power BI para explorar indicadores, causas dos acidentes e padrões de mortalidade por meio de dashboards interativos.


🚦 Análise de Acidentes em Rodovias Federais — Brasil (2017)

Este projeto tem como objetivo analisar acidentes de trânsito ocorridos nas rodovias federais do Brasil em 2017, utilizando Power BI para modelagem, análise e visualização dos dados.

A proposta foi transformar dados brutos em informações claras, visuais e orientadas a insights, permitindo identificar padrões, fatores de risco e possíveis oportunidades para melhoria da segurança viária.

Além da análise de dados, o projeto também buscou aplicar conceitos de storytelling com dados, design de dashboards e experiência do usuário (UX), tornando as informações acessíveis não apenas para analistas de dados, mas também para gestores e tomadores de decisão.

🎯 Objetivos do Projeto

O projeto foi desenvolvido com alguns objetivos principais:

Explorar dados de acidentes rodoviários para identificar padrões e tendências

Construir indicadores estratégicos (KPIs) para monitoramento

Aplicar modelagem de dados e medidas em DAX

Criar um dashboard interativo e intuitivo

Demonstrar como a análise de dados pode apoiar decisões relacionadas à segurança no trânsito

🛠️ Ferramentas Utilizadas

Power BI — Modelagem de dados, criação de medidas em DAX e visualização

DAX (Data Analysis Expressions) — Criação de indicadores e métricas

VS Code + IA como copiloto — Apoio na construção e otimização de medidas

Figma — Criação do layout visual e estrutura do dashboard

Power Query — Tratamento e transformação dos dados

Durante o desenvolvimento, a integração do VS Code com IA ajudou a acelerar a criação de algumas medidas. No entanto, todas as fórmulas foram revisadas e ajustadas manualmente, garantindo a consistência dos indicadores utilizados no dashboard.

💡 Principais Insights Encontrados

A análise revelou alguns padrões importantes nos acidentes rodoviários analisados:

📈 Variação ao longo do ano

Junho apresentou a maior taxa de mortalidade (~5,7%), indicando possível influência de fatores sazonais ou aumento de fluxo em determinadas épocas.

👤 Perfil das vítimas

Homens representam cerca de 79% das vítimas fatais, mostrando forte predominância masculina nos acidentes com morte.

🌞 Período do dia

A maior concentração de acidentes ocorre durante o dia, possivelmente devido ao maior fluxo de veículos nas rodovias nesse período.

⚠️ Principais causas

Entre as principais causas de acidentes estão:

Falta de atenção na condução

Velocidade incompatível

Ingestão de álcool

Desobediência às normas de trânsito

A falta de atenção na condução aparece como um dos fatores mais recorrentes associados aos acidentes.

🏍️ Tipo de veículo

Embora automóveis concentrem o maior número absoluto de acidentes,

motocicletas apresentam taxa de mortalidade proporcionalmente maior, indicando maior vulnerabilidade desse tipo de veículo.

📍 Distribuição geográfica

Alguns estados apresentam maior concentração de acidentes no período analisado, com destaque para:

Minas Gerais

Paraná

Santa Catarina

Essas regiões concentram grande volume de ocorrências nas rodovias federais.

🚦 Possíveis Soluções Baseadas nos Dados

A análise permite levantar algumas ações que poderiam contribuir para a redução de acidentes:

1️⃣ Campanhas de conscientização

Programas voltados para atenção na condução, já que a distração aparece como uma das principais causas.

2️⃣ Políticas específicas para motociclistas

Medidas como:

incentivo ao uso correto de equipamentos de proteção

campanhas de direção defensiva

treinamentos para motociclistas

Isso se justifica pela maior taxa de mortalidade associada às motocicletas.

3️⃣ Fiscalização em horários de maior fluxo

Como grande parte dos acidentes ocorre durante o dia, políticas de fiscalização e controle de velocidade podem ser intensificadas nesses períodos.

4️⃣ Ações regionais

Estados com maior número de ocorrências podem receber:

melhorias na infraestrutura viária

campanhas regionais de segurança

monitoramento mais intensivo.

5️⃣ Uso contínuo de dados para monitoramento

Dashboards como este podem apoiar gestores públicos e órgãos de trânsito, permitindo acompanhar indicadores em tempo real e identificar rapidamente mudanças no padrão de acidentes.
